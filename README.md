# Partner48
口袋48 成员直播 PC版
# 特色🎯
- 安装即用，免ffmpeg安装
- 成员直播/录播多开
- 支持win/mac版本
- 直播弹幕需要设置网易云信IM令牌，直播弹幕复用48tools代码，该项目项目中大佬提供了一个令牌：https://github.com/duan602728596/48tools/blob/main/packages/48tools/src/pages/PlayerWindow/sdk/appKey.mjs
